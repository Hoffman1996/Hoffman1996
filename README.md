# About Me:
### Hi there 👋
I'm Yuval, a B.Sc. Software Engineering graduate from Afeka Academic College of Engineering.

I build end-to-end **Computer Vision systems**, **Machine Learning pipelines**, and **backend services** in Python, Java, and C.

My capstone project is a real-world window frame measurement system — combining a custom-trained **YOLOv11 segmentation model** with **ChArUco board calibration** to measure physical dimensions from a single smartphone photo, achieving **sub-3% average error** and **98.8% detection rate** across 154 test images.

I enjoy solving complex technical challenges, working close to the hardware, and turning research ideas into working systems.
